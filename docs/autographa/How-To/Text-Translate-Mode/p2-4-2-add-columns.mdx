---

- Find the **Columns** icon <img src="/autographaAssetsV2/how-to/add-cloumn.png"  width="50px" alt="add column"/> on the top right corner of the screen. 
- Click on it to change the number of columns shown on the screen.
- **Autographa** supports up to three columns open at a time (including the editing column). 

<video controls src="/autographaAssetsV2/how-to/add-columns.mov" width="100%" type="video/mov"/>

Each column can load different reference resources inside sections. Each column can have up to three sections.
Reference resources loaded inside the sections are updated by clicking on different verses in the editing column.
